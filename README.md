## Features
- Modern UI with Ant Design Vue
- Add, remove tasks
- Save all change and tasks in cookies or on Database!
- Sort you list by finished or not
- uhhhhhhhh

## Project setup and run

#### For v2.0.0 up
1. `npm install`
2. Copy `.env.example` to `.env` and fill in your database information
3. `npm run dev`

#### For v1.0.0
1. `npm install`
2. `npm run dev`
